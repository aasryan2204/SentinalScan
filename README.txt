# SentinelScan

SentinelScan is a modular Python-based network scanner built from scratch to demonstrate networking, cybersecurity, and software engineering principles.

## Planned Features

- TCP Port Scanning
- Service Detection
- Banner Grabbing
- Multithreaded Scanning
- CSV Reports
- JSON Reports
- HTML Reports
- Logging
- Configuration Profiles

## Technologies

- Python
- Socket Programming
- Rich
- Git
- GitHub
